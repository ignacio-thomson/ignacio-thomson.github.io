Acerca de mi


Soy un Desarrollador Full Stack Java, actualmente en nivel Junior. Egresé del Bootcamp Desafio Latam, donde fui parte del curso intensivo Java, con el objetivo final de desarrollar aplicaciones de carácter empresarial. Utilizamos herramientas como Eclipse IDE, Intellij IDEA, Netbeans IDE, DBeaver, etc. Trabajamos con librerías para el Desarrollo Web como Bootstrap, MDBootstrap y JQuery. En el aspecto Java, utilizamos el Framework de Springboot para crear nuestras aplicaciones, donde nos centramos principalmente en generar un CRUD ordenado y estructurado. Utilizamos las bases de datos de Oracle, PostgreSQL y MySQL. Actualmente estoy estudiando por cuenta propia, para poder trabajar de forma apropiada en el Front-end con ReactJS y AngularJS.

Logistiqal

Sitio web inspirado en manejo de bódega para locales comerciales

Logistiqal fue un proyecto orientado a imitar una aplicación donde se pudiera administrar la bódega de un local comercial. Al igual que en el proyecto IMDb, se puede realizar un CRUD con los productos, es decir, crear, leer, actualizar o eliminarlos de la base de datos. También posee una barra de búsqueda que devuelve como resultado todos los productos que contengan la palabra utilizada como criterio de búsqueda.
