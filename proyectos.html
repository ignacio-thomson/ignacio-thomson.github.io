Web APP - ReactJS & NodeJS - Express

Aplicación Web con arquitectura RESTful, con su Back-end desarrollado en NodeJS & Express. Front-end desarrollado en el Framework ReactJS. Se utiliza sistema de autenticación por JWT para validar el acceso a los distintos Endpoints. Se puede manejar el modo claro u oscuro. Simula ser un centro de administración para la empresa ficticia Coreware para añadir, editar, eliminar y leer, componentes, bodegas o distribuidores. Se utilizó la base de datos MongoDB. Se realizó el despliegue en Vercel (Front-end) y Railway (Back-end).

Logistiqal

Sitio web inspirado en manejo de bódega para locales comerciales

Logistiqal fue un proyecto orientado a imitar una aplicación donde se pudiera administrar la bódega de un local comercial. Al igual que en el proyecto IMDb, se puede realizar un CRUD con los productos, es decir, crear, leer, actualizar o eliminarlos de la base de datos. También posee una barra de búsqueda que devuelve como resultado todos los productos que contengan la palabra utilizada como criterio de búsqueda.
